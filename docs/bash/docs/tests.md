# tests/ Overview

## Purpose
Test suite for validating configuration, functions, and integrations.

## Contents
- run_all_tests.sh (project root): Run the full test suite.
- run_tests_with_spinners.sh: Run tests with spinner output.
- test_aliases.sh: Tests for alias definitions and loaders.
- test_autocomplete.sh: Tests for autocomplete module.
- test_readline.sh: Tests for readline completion settings.
- test_docker.sh: Tests for docker helpers.
- test_filesystem.sh: Tests for filesystem helpers.
- test_integration.sh: Tests for integrations.
- test_make_dir_tree.sh: Tests for the tree generation script.
- test_modules.sh: Tests for module loading.
- test_music.sh: Tests for music helpers.
- test_themes.sh: Tests for theme logic.
- test_utils.sh: Tests for core utilities.
- logs/: Test logs output directory.
- OVERVIEW.md: This file.

## Usage Notes
- Run tests via tests/run_tests_with_spinners.sh or run_all_tests.sh in the project root.
- Logs are stored in tests/logs.
