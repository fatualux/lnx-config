# docs/ Overview

## Purpose
Project-wide documentation, including guides, structure snapshot, and changelog.

## Contents
- README.md: Summary documentation for the project and links to each directory overview.
- CHANGELOG.md: Semantic-versioned change log of features and fixes.
- STRUCTURE.md: Auto-generated directory tree snapshot.
- **completion.md**: Smart completion system guide with usage, caching, and troubleshooting.
- config.md: Configuration system overview.
- core.md: Core utilities documentation.
- functions.md: Functions module documentation.
- integrations.md: Integration modules guide.
- aliases.md: Aliases system overview.
- themes.md: Theme system and customization.
- tests.md: Testing framework and guidelines.
- OVERVIEW.md: This file.

## Usage Notes
- Regenerate STRUCTURE.md after adding/removing files using scripts/make-dir-tree.sh.
- Keep CHANGELOG.md updated for each change set.
- See [completion.md](completion.md) for tab-completion features and configuration.
