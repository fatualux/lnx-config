# docs/ Overview

## Purpose
Project-wide documentation, including summary, structure snapshot, and changelog.

## Contents
- README.md: Summary documentation for the project and links to each directory overview.
- CHANGELOG.md: Semantic-versioned change log of features and fixes.
- STRUCTURE.md: Auto-generated directory tree snapshot.
- OVERVIEW.md: This file.

## Usage Notes
- Regenerate STRUCTURE.md after adding/removing files using scripts/make-dir-tree.sh.
- Keep CHANGELOG.md updated for each change set.
