# functions/ Overview

## Purpose
Category-based shell functions loaded by main.sh for feature organization.

## Contents
- aliases/: Helpers for listing and managing aliases.
- development/: Git and editor helpers.
- docker/: Docker management utilities.
- filesystem/: Cleanup and filesystem helpers.
- music/: MPV playlist helpers.
- OVERVIEW.md: This file.

## Usage Notes
- All .sh files within each subdirectory are sourced automatically.
- Keep new functions in the appropriate category for clarity.
